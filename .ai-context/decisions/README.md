# Architecture Decision Records (ADR)

Store major technical decisions in this folder.

## Naming

Use incremental numbering:

- `001-short-title.md`
- `002-short-title.md`

## Suggested Template

Each ADR should include:

1. Context
2. Decision
3. Consequences
4. Alternatives considered

## Scope

Record decisions that affect architecture, data flow, performance, reliability, cost, or team workflow.
